## Run Locally

Clone the project

```bash
  git clone https://github.com/AkhrorSoliev/frontend-quiz-app
```

Go to the project directory

```bash
  cd frontend-quiz-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
